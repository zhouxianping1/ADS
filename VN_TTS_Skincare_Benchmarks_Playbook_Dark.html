① 速查键值｜Quick Keys

优秀投放（放量复用）Go Big

  • CTR > 1.5%，CPC < $0.22，VTR≥3s > 38%
  • PDP：加购 > 15%、下单 > 3.5%

优先修创意（封面+前三秒）

  • CTR < 0.5% 或 VTR≥3s < 30%
  • 钩子=封面大字=前3秒台词；人脸近景+粗字幕

优先修定向（受众/语言/版位）

  • CPM > $6 且 CTR 低 → 放宽语言为 All；兴趣≤2个宽词（Chăm sóc da / Skin Care）

优先修 PDP（承接卖点/优惠）

  • CTR ≥ 1% 但 加购 < 10% 或 下单 < 2%
  • 首屏放 券/赠/包邮 + 卖点图 + 评价锚点；默认选中主推 SKU

② 怎么用｜Cách dùng

  1. 用你账户最近 24–72h 的每条广告数据对照下面表格定位到“差/及格/良好/优秀”。
  2. 按“建议动作”执行：停／留／小幅加预算（≤+20%）／复用放大。
  3. 结合“年龄段预期”判断是否要换 钩子 或 诉求点。

⑤ 排查步骤｜Quy trình soát lỗi（4–6小时一轮）

  1. 看 CTR：<0.5% 直接换钩子（封面/前3秒）。
  2. 看 CPC：>$0.35 同时检查兴趣是否过窄、语言是否仅越语（改为 All）。
  3. 看 VTR≥3s：<30% → 镜头更近、人脸更大、文案更短；3秒内给出“2泵→5点→轻拍”。
  4. 看 PDP 加购/下单：<10% / <2% → PDP 首屏放 券&赠&包邮、评价截图、功效动图；把置顶评论切换成“活动版”。
  5. 达到良好/优秀：逐日 +10–20% 放量；每组保留 3–5条 活跃广告。

⑥ 常见问题｜FAQ

Q1：CPM 很低但 CTR 也低？
→ 广泛定向给的量不精准；提高 创意相关性（封面大字 = 前3秒文案），把 商品卡标题≤30字 改成同一关键词。

Q2：CTR 高但下单率低？
→ PDP 没有“承接同款卖点”。把 封面钩子（如“3秒提亮/7天淡印/抚纹3秒”）做成 主图角标 与 前两张详情图大字，并默认选中主推 SKU。

Q3：CPC 忽高忽低？
→ 每次只改 一个变量，并控制 预算浮动 ≤20%/次，避免学习反复重置。

⑦ 注意｜Lưu ý

  • 统一四档标准后，可快速定位问题在 创意 / 定向 / PDP。
  • 遇到 CTR 高但 CPC 也高 的情况，往往是 CPM 高 + 无效点击：优先放宽语言与兴趣，再优化前3秒。
  • 转化低多半在 PDP 或优惠：把 口令/赠品/券 搬到 商品标题首屏 + 主图角标，并在置顶评论复述一次。
